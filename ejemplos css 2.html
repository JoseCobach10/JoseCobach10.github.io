la red informática mundial

En informätica, la World wide Web (M) o red informática mundial es un sistema de distribución de documentos de hipertexto o hipermedios interconectados y accesibles via Internet.

Con un navegador web, un usuario visualizo sitios web compuestos de páginas web que pueden contener texto, inágenes, vídeos u otros contenidos mundiales, y navegar a traves de esas paginas usando hiperenlaces.
